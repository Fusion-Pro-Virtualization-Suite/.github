# VMware Fusion Software — Professional OS Virtualization for Mac

## About VMware Fusion Software

VMware Fusion Software provides professional virtualization for running Windows, Linux, and Kubernetes clusters on Intel and Apple Silicon Mac hardware. VMware Fusion Software combines Unity mode, 3D graphics acceleration, and multi-monitor display in one hypervisor platform. VMware Fusion Software supports USB 3.1 passthrough, drag-drop file sharing, and snapshot management. VMware Fusion Software enables encrypted VMs, linked clones, and vSphere remote connectivity. VMware Fusion Software includes command-line tools (vmrun), network simulation, and REST API control. VMware Fusion Software is trusted by software developers, IT administrators, DevOps engineers, QA teams, and enterprise Mac users worldwide.

## Quick Reference for VMware Fusion Software

| Operation | VMware Fusion Software Action |
|-----------|------------------------------|
| New VM | File → New → Select OS installer |
| Start VM | Click Play button |
| Enter Unity mode | View → Enter Unity |
| Take snapshot | Snapshots → Take Snapshot (Cmd+Shift+S) |
| Revert to snapshot | Snapshots → Select → Revert to Snapshot |
| Enable encryption | VM Settings → Encryption → Set password |
| Create linked clone | Virtual Machine → Clone → Linked clone |
| Connect to vSphere | File → Connect to Server → Add ESXi/vCenter |
| Passthrough USB device | VM → USB → Select device |
| Open vmrun terminal | Applications → VMware Fusion → vmrun |
| Simulate network | VM Settings → Network → Advanced → Simulation |
| Export VM to OVF | File → Export to OVF |
